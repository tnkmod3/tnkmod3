A Short Tutorial on VLookUp and IfError Functions!

There are a lot of different functions in Microsoft Excel. One of the many popular ones is the VLookUp function.

With this function it allows you to type in a keyword, reference a column of data, and return back a value based on the previous values.

It's extremely useful when dealing with products and prices, names and numbers, as well as salaries and tax rates. The things to remember when using the VLookUp function are:

  1. Always use false if you're looking for an exact match
  2. Be sure to name the table range you want to use
  3. VLookUp can cascade down for multiple uses
  4. Use IfError if you want your data to look smoother
